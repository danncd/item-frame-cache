<h1>Item Frame Cache</h1>
<p>This mod was designed to solve a multiplayer server issue where item frames disappear after walking a few blocks away (When simulation distance is significantly lower than client's side). By caching item frames to local disk, the mod ensures that item frames remain visible regardless of the server unloading data.</p>

<p>You can download the mod in the <a href="https://github.com/danncd/item-frame-cache/releases">releases page</a>.</p>
