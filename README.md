<h1>Item Frame Cache</h1>
<p>This mod was designed to solve many multiplayer server issues where item frames disappear after walking a few blocks away. By caching item frames to local disk, the mod ensures that item frames remain visible regardless of the server unloading data.</p>

<p>You can download the mod in the <a href="https://github.com/danncd/item-frame-cache/releases">releases page</a>.</p>
